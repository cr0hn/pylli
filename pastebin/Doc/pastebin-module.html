Unofficial pastebin.com API for Python by Mario Vilas (mvilas at gmail dot com)

Unlike the official API, this one doesn't require an API key.

http://breakingcode.wordpress.com/2012/01/27/pasting-anonymously-to-pastebin-com/

http://breakingcode.wordpress.com/2010/03/06/using-the-pastebin-api-with-python/

Usage examples:

   >>> from pastebin import submit
   >>>
   >>> # Submit a simple text message.
   ... url = submit("This is a test message.")
   >>> print url
   http://pastebin.com/G4YWsB1a
   >>>
   >>> # Submit a simple "hello world" code in Python.
   ... # Paste expires in 10 minutes.
   ... url = submit("print "Hello world!"",
   ...               paste_format="python",
   ...               paste_expire_date="10M")
   >>> print url
   http://pastebin.com/rKEVDWHP
   >>>
   >>> # Submit Apache's logs as a private paste.
   ... # Give the paste a title.
   ... url = submit(open('/var/log/apache/access.log', 'r').read(),
   ...              paste_format="apache",
   ...              paste_private=True,
   ...              paste_name="My Apache access logs")
   >>> print url
   http://pastebin.com/kfepMRLh
   >>>

Function Details [hide private]

submit(paste_code, paste_name=None, paste_private=None, paste_expire_date=None, paste_format=None)

 

Submit a code snippet to Pastebin.

Parameters:
  • paste_code (str) - Code or text to send to http://pastebin.com.
  • paste_name (str) - (Optional) Name of the author of the paste. Default is to paste anonymously.
  • paste_private (bool) - (Optional) True if the paste is private (only visible with the link), False if it's public (indexed and searchable). The Pastebin FAQ (http://pastebin.com/faq) claims private pastes are not indexed by search engines (aka Google).
  • paste_expire_date (str) - (Optional) Expiration date for the paste. Once past this date the paste is deleted automatically. Valid values are found in the Pastebin.paste_expire_date class member. If not provided, the paste never expires.
  • paste_format (str) - (Optional) Programming language of the code being pasted. This enables syntax highlighting when reading the code in http://pastebin.com. Default is no syntax highlighting (text is just text and not source code).
Returns: str
Returns the URL to the newly created paste.
Raises:
  • PastebinError - The Pastebin API has returned an error message.
  • IOError - An error occurred when contacting the Pastebin web application. This is typically caused by network errors.
